Data type

A data type is a classification of data which tells the compiler or interpreter how the programmer intends to use the data.

Input & Output

An input is data that a computer receives. An output is data that a computer sends.

Operator

An operator, in computer programing, is a symbol that usually represents an action or process.

Conditional Logic

Conditional statements are used through the various programming languages to instruct the computer on the decision to make when given some conditions.

Switch Statement

The switch statement evaluates its expression, then executes all statements that follow the matching case label.

Increment & Decrement

The process of increasing or decreasing a numeric value by another value.

Loop

A loop is a sequence of instruction s that is continually repeated until a certain condition is reached.

Array

An array is a data structure, which can store a fixed-size collection of elements of the same data type.

Function

A function is simply a “chunk” of code that you can use over and over again, rather than writing it out multiple times.

Pointer

In computer science, a pointer is an object in many programming languages that stores a memory address.
